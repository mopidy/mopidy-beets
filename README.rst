Mopidy-Beets
=================

Mopidy http://www.mopidy.com/ extension for playing music from
Beets http://beets.radbox.org/ via beets web extension.

Usage
-----

Install by running::

    sudo pip install mopidy-beets


Setup your beets host

    [beets]
    hostname = 127.0.0.1
    port = 8888
